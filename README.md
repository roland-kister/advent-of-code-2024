# [Advent of Code 2024](https://adventofcode.com/2024)

Save the individual puzzle inputs to `./inputs` directory with leading zero in the file name, e.g. `01.txt`, or `13.txt`:

```txt
./inputs/
    ├─01.txt
    ├─02.txt
    ...
    └─25.txt
```

Execute the solution:

```shell
go run .
```

Or run the tests:

```shell
go test ./...
```
