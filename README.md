# advent-of-code-2024

Save inputs to `./inputs` with leading zero in file name, e.g. `01.txt`, or `13.txt`:

```txt
./inputs/
    ├─01.txt
    ├─02.txt
    ...
    └─25.txt
```

Execute the solution with the following command:

```shell
go run .
```

Or to run the tests execute:

```shell
go test -v ./...
```
